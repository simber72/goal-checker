# 🛠️ AnBxJ library
The ```AnBx``` library belongs to the AnBx framework https://paolo.science/anbx/ and it is used to
derive the Java-based implementation of a protocol from its Alice&Bob specification.
The Java-based implementation of a protocol also relies upon the ```AnBx``` library.

The experiments have been carried out with the AnBx Compiler version 2025.09.
